Open Your Terminal

Open Terminal (Mac/Linux) or PowerShell (Windows). You can use VS Code's integrated terminal too.

On Mac, press Cmd + Space and type "Terminal". On Windows, search for "PowerShell".

Run the Magic Command

Copy and paste this command into your terminal:

gh copilot skill https://raw.githubusercontent.com/kody-w/rapp-installer/main/skill.md

Press Enter to run it.

Follow Copilot's Instructions

Copilot will guide you through the entire setup process. It will:

  • Ask you questions about your setup
  • Clone the RAPP repository
  • Create Azure resources for you
  • Configure everything automatically

When prompted to login to Azure, a browser window will open. Sign in with your Azure account.

Start Building Agents!

Once setup is complete, Copilot will start RAPP locally. You can then:

  • Open http://localhost:7071 in your browser
  • Upload transcripts to generate AI agents
  • Deploy to Microsoft Copilot Studio
